# Discord Bot with Python
This is a small project made purely for fun and work with a new library (Discord.Py library).

# Features:
- Adding/Removing roles based in reactions
- Can now play BlackJack (Card Game)

# How to use:
- Download the DiscordPyBot folder
- Install all the requirements (requeriments.txt) by running this in your terminal:
  ```python
  pip install -r \requirements.txt
  ```
- In the https://discord.com/developers/applications create a new application
- Click on the Bot tab and scroll down, enable the Server Members Intent and the Message Content Intent
- Put your Bot discord token in the .env file
- Run the main.py file and be happy!
